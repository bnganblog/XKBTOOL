系统信息 ​

功能概述 ​

系统信息页面展示当前电脑的硬件信息和资源使用情况，帮助用户快速了解系统状态。

硬件检测 ​

项目说明
CPU型号、核心数、线程数、频率
内存总容量、已用、可用
GPU型号、显存、驱动版本
硬盘型号、容量、分区信息
外设鼠标、键盘等外设信息

用量图表 ​

支持两种图表样式，可在 设置 → 外观 中切换：

环形图 ​

类似 Windows 任务管理器的环形进度图，直观显示 CPU、内存、GPU 的使用率。

条形图 ​

水平进度条样式，紧凑直观，适合在小窗口中查看。

一键烤鸡 ​

集成烤鸡工具入口，快速启动稳定性测试：

  • CPU 烤鸡
  • GPU 烤鸡
  • 内存测试

实时刷新 ​

系统信息页面支持实时刷新，数据自动更新，无需手动刷新。
