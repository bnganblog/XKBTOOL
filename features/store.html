工具商店 ​

功能概述 ​

工具商店提供插件扩展功能，可以一键安装额外的工具组件。

安装插件 ​

  1. 点击左侧导航栏「工具商店」
  2. 浏览可用插件列表
  3. 点击「安装」按钮开始下载
  4. 下载完成后自动解压到应用目录
  5. 左侧导航栏自动更新，显示新安装的工具

插件列表 ​

ProxyTools ​

基于 Mihomo 内核的代理工具插件，提供完整的代理管理功能。

  • 大小: ~21 MB
  • 功能: 代理管理、配置编辑、系统代理设置
  • 依赖: 无（内含 Mihomo v1.19.11）

下载机制 ​

  • 优先使用 GitHub 加速代理下载
  • 下载失败自动回退到 GitHub 直连
  • 支持在 设置 → 更新 → 下载代理 中配置自定义代理地址
  • 下载进度实时显示在窗口底部

已安装检测 ​

  • 已安装的插件显示「打开」按钮（蓝色）
  • 未安装的插件显示「安装」按钮（绿色）
  • 自动检测 ProxyTools/mihomo/mihomo.exe 是否存在

下载代理 ​

如果 GitHub 下载速度较慢，可以在 设置 → 更新 → 下载代理 中配置加速代理：

代理地址说明
https://ghfast.top/默认加速代理
https://ghproxy.com/备用代理
留空直连 GitHub

提示

代理地址会自动保存，下次下载时自动使用。
