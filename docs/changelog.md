# 更新日志

## v1.0.0

### 新功能

- **系统信息** — CPU、内存、GPU、硬盘型号实时检测
- **网络工具** — IP 查询、分流测试、网速测试
- **代理工具** — 内置 Mihomo 代理引擎，一键启停
- **工具商店** — 一键安装插件扩展
- **收藏夹** — 工具收藏与分类管理
- **毛玻璃主题** — 浅色/深色亚克力毛玻璃效果
- **系统托盘** — 最小化到托盘，右键菜单快速操作
- **自动检查更新** — GitHub Release 更新检测
- **配置导入导出** — 工具配置 JSON 导入导出
- **动态目录同步** — 自动扫描 tools/ 目录

### 技术栈

- WinUI 3 + Windows App SDK 1.x
- .NET 10
- C# 14
- Mihomo v1.19.11
