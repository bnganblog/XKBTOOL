# 快速开始

## 下载与安装

1. 从 [GitHub Release](https://github.com/bnganblog/xkbtool/releases/latest) 下载最新版 `XKBToolbox.zip`
2. 解压到任意目录（建议非系统盘）
3. 双击运行 `ToolboxWinUI.exe`

::: tip 首次运行
如果系统未安装 .NET 10 Desktop Runtime，会自动提示安装。
:::

## 界面导航

启动后默认显示 **系统信息** 页面，左侧为导航栏：

| 导航项 | 功能 |
|--------|------|
| 系统信息 | 查看 CPU、内存、GPU、硬盘等硬件信息及用量 |
| 收藏夹 | 快速访问已收藏的工具 |
| CPU工具 | CPU 相关检测与优化工具 |
| 显卡工具 | GPU 相关检测与优化工具 |
| 内存工具 | 内存检测与优化工具 |
| 硬盘工具 | 硬盘检测与管理工具 |
| 外设工具 | 鼠标、键盘等外设工具 |
| 烤鸡工具 | 稳定性测试工具 |
| 网络工具 | IP 查询、分流测试、网速测试 |
| 其他工具 | 其他实用工具 |
| 工具商店 | 下载安装插件扩展 |

## 使用工具

每个工具以卡片形式展示，包含：

- **图标** — 工具标识
- **名称** — 工具名称
- **描述** — 工具简介
- **⭐ 收藏** — 点击星标收藏工具
- **⚙ 设置** — 编辑工具信息
- **打开** — 启动工具

## 添加自定义工具

1. 将工具文件放入 `tools/` 目录
2. 点击左侧导航栏对应分类
3. 工具会自动出现在卡片网格中

也可以在 设置 → 工具管理 中导入/导出工具配置。
