# 代理工具

## 功能概述

代理工具基于 Mihomo 内核，提供完整的代理管理功能，支持一键启停、配置管理、系统代理设置。

## 核心功能

### 一键启停

- 点击「启动」按钮启动代理
- 点击「停止」按钮停止代理
- 状态指示灯实时显示运行状态（绿色=运行中，红色=已停止）

### 配置管理

- **上传配置** — 支持 `.yaml`/`.yml` 格式配置文件
- **编辑配置** — 内置文本编辑器，在线修改配置
- **保存配置** — 自动追加必要字段，保存后重启代理生效

### 系统代理

- 启动代理时自动设置 Windows 系统代理
- 停止代理时自动清除系统代理
- 支持 HTTP/SOCKS5 代理协议

## 端口说明

| 端口 | 用途 |
|------|------|
| 7890 | 混合代理端口（HTTP/SOCKS5） |
| 9097 | REST API 端口 |
| 19090 | Dashboard 面板端口 |

## Dashboard

代理运行后，可通过浏览器访问 MetaCubeXD 面板：

```
http://127.0.0.1:19090/ui
```

面板功能：
- 节点管理
- 规则管理
- 连接监控
- 流量统计

## 内核信息

- **Mihomo 版本**: v1.19.11
- **支持协议**: VMess, VLESS, Trojan, Shadowsocks, Hysteria, TUIC, WireGuard 等
- **配置格式**: YAML
