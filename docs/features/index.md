# 功能概览

显卡吧工具箱集成了丰富的硬件工具和系统功能，以下是主要功能模块介绍。

## 核心功能

### 系统信息

实时监控硬件状态，包括 CPU、内存、GPU、硬盘等信息，支持环形图和条形图两种展示方式。

[了解更多 →](./system)

### 网络工具

集成 IP 查询、分流测试、网速测试三大功能模块，支持多源数据对比和实时图表展示。

[了解更多 →](./network)

### 代理工具

内置 Mihomo 代理引擎，支持一键启停、配置管理、系统代理设置，附带 MetaCubeXD 面板。

[了解更多 →](./proxy)

### 工具商店

一键安装插件扩展，自动检测已安装状态，支持 GitHub 加速下载。

[了解更多 →](./store)

## 辅助功能

| 功能 | 说明 |
|------|------|
| 收藏夹 | 支持工具收藏，常用工具一键直达 |
| 毛玻璃主题 | 浅色/深色亚克力毛玻璃效果 |
| 动态目录同步 | 自动扫描 tools/ 目录 |
| 配置导入导出 | 工具配置 JSON 导入导出 |
| 自动检查更新 | GitHub Release 更新检测 |
| 系统托盘 | 最小化到托盘，右键菜单快速操作 |
