常见问题 ​

启动问题 ​

双击 exe 无反应 ​

  1. 检查是否已安装 .NET 10 Desktop Runtime
  2. 以管理员身份运行
  3. 检查杀毒软件是否拦截

提示缺少 .NET 运行时 ​

从 Microsoft 官网 下载并安装 .NET 10 Desktop Runtime。

工具管理 ​

工具不显示 ​

  1. 确认工具文件在 tools/ 目录下
  2. 检查文件扩展名是否为 .exe、.lnk 或 .bat
  3. 重新进入对应分类页面

如何添加自定义工具 ​

将工具文件放入 tools/ 目录下的对应分类文件夹中，或在 设置 → 工具管理 中导入 JSON 配置。

代理工具 ​

代理工具不见了 ​

  1. 检查 ProxyTools/mihomo/mihomo.exe 是否存在
  2. 如不存在，在工具商店重新安装
  3. 如存在但仍不显示，重启应用

代理无法启动 ​

  1. 检查端口 7890 和 9097 是否被占用
  2. 检查配置文件格式
  3. 查看日志文件

系统代理未生效 ​

  1. 检查 Windows 设置 → 网络和 Internet → 代理
  2. 确认代理已启动（状态指示灯为绿色）
  3. 尝试重启代理

下载问题 ​

插件下载失败 ​

  1. 检查网络连接
  2. 在 设置 → 更新 → 下载代理 中配置加速代理
  3. 尝试手动下载 ProxyTools.zip 并解压到应用目录

下载速度慢 ​

  1. 配置 GitHub 加速代理
  2. 使用 VPN 或代理工具
  3. 手动从 GitHub 下载

界面问题 ​

毛玻璃效果不显示 ​

  1. 确认 Windows 版本为 Windows 11 或 Windows 10 1809+
  2. 检查系统设置中是否启用了透明效果
  3. 尝试切换主题

界面错乱 ​

  1. 尝试切换主题
  2. 重启应用
  3. 恢复默认设置

其他问题 ​

配置文件在哪里？ ​

%APPDATA%\ToolboxWinUI\
├── settings.json    # 应用设置
├── tools.json       # 工具配置
└── proxy/           # 代理配置

如何完全卸载？ ​

  1. 删除应用目录
  2. 删除 %APPDATA%\ToolboxWinUI\ 目录
