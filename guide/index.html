简介 ​

显卡吧工具箱是一款面向 PC 硬件爱好者与 DIY 玩家的 Windows 桌面工具集，将各类硬件检测、性能测试、系统优化工具整合到统一的现代化界面中。

特性 ​

  • 智能工具管理 — 自动扫描 tools/ 目录，以卡片网格展示，支持收藏、打开
  • 硬件监控 — CPU、内存、GPU、硬盘型号实时检测，含环形用量图表
  • 网络诊断 — IP 地址多源查询、分流测试（DNS 定位 + Ping）
  • 毛玻璃主题 — 浅色/深色亚克力毛玻璃效果，支持实时切换
  • 动态目录同步 — 每次启动自动扫描，新增删除即时生效
  • 代理工具 — 内置 Mihomo 代理引擎，一键启停，配置管理
  • 工具商店 — 一键安装插件扩展，支持 GitHub 加速下载

技术栈 ​

  • 框架: WinUI 3 + Windows App SDK 1.x
  • 运行时: .NET 10
  • 语言: C# 14
  • 系统: Windows 10/11
